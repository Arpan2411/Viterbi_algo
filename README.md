# Viterbi_algo
A simple implementation of Hidden Markov Model in parts of speech tagging using Viterbi alogrithm.

Since the Emission probability table, transition probability table requires a lots of data to measure the probability value but here I have taken some value based on assumption, thats why the output given by it is not 100 percent accurate.
This is a demonstration of the Viterbi Algorithm, actual application of this algo needs optimization , and also required a lots of data to train to get the probebility value.
